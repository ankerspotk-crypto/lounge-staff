3. Google My Business API からのメール取得

3.1 仕組み

``` 【自動プロセス】 1. 新規投稿が発生 2. 店舗名で Google Places API 検索 3. Google My Business データを取得 ├─ 店舗名（確定） ├─ 住所（確定） ├─ 電話番号（確定） ├─ メールアドレス（50-70%の確率） └─ オーナー名（30-40%） 4. メール取得成功 → SendGrid キューに追加 5. 即座にメール配信開始 【実装】 API 呼び出し：Google My Business API / Places API フロー制御：AWS Lambda（無料枠で運用可） データベース：DynamoDB（投稿 ↔ 店舗紐付け） ```
