4. フェーズ別戦略

Phase 1：Month 1-2（テスト運用）

  • 投稿獲得：TikTok（自分たちで投稿）+ Google クロスポスト + 口コミのみ
  • 広告費：¥50,000/月（インフルエンサー報酬は後払い or 成果報酬化）
  • 店舗営業：まず 10店舗への直営業（デモデータ作成用）
  • 投稿数目標：3,000-4,000件（キャンペーン2倍ポイントで急かす）
  • 目標達成：¥30K のプラン契約 × 1-2店 → キャッシュ確保

Phase 2：Month 3-4（本格営業）

  • 投稿獲得：Google クロスポスト本格化（API 統合完了）
  • 店舗営業：掲載 100 店舗を対象に LIGHT プラン営業開始
  • 目標契約数：¥30K × 3-5店舗（月額 ¥3K-5K）
  • 調査員開始：Lv.2 見習い調査員の第一期選抜（5-10人）
  • 月間売上：¥80K-150K → 運用費 ¥100K をカバー開始

Phase 3：Month 5-6（自立化）

  • 投稿獲得：自然な口コミ + 既存ユーザーによる継続投稿
  • 店舗営業：月 20-30 件の営業で月 ¥200K-300K の新規契約
  • 調査員数：Lv.2 50 人、Lv.3 5 人稼働
  • 月間売上：¥350K-400K
  • 完全黒字化：月 +¥200K-300K の利益で再投資開始

③ Month N の調査員報酬は Month N+1 に支払い
④ 結果：常に前月の利益で当月の報酬をカバー
